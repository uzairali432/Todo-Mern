# 📝 Todo MERN App

A full-stack **Todo Application** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.  
This project allows users to create, update, delete, and manage daily tasks with a clean and responsive user interface.

---

## 🚀 Features

- ✅ Add new todos
- ✏️ Edit existing todos
- 🗑️ Delete todos
- 📌 Mark tasks as completed
- 🔄 Real-time UI updates
- 🌐 RESTful API integration
- 📱 Responsive design

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- CSS / Bootstrap (if used)

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose ODM)
